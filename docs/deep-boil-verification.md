- interval Picard enclosures for Rössler, Lorenz, and Thomas;
- canonical Z₂ and truncated U(1) Hamiltonian construction;
- Hermiticity and finite-gap audits;
- fail-closed research-claim manifests;
- generated Experience/Experiment browser bundle;
- package installation and Foundry registry discovery;
- existing GaugeGap, FlowGap, CurveRank, quantum, proofpack, and audit lanes.
