# Lagrangian Forge

Lagrangian Forge is the eighth Foundry Experience scene. It converts the visual density of an expanded Standard Model blackboard into a canonical field catalog, interaction hypergraph, tree-level electroweak calculator, and structural audit.

The artistic transcription is visual inspiration only. The source of truth is the compact Standard Model sector decomposition implemented in `src/gaugegap/standard_model_catalog.py`.

## Run

```bash
foundry run foundry-experience-v2
# focused alias
foundry run lagrangian-forge
```

Open `site/foundry-experience/index.html`.

For this scene the projection selector chooses between:

- interaction graph;
- equation wall;
- symmetry breaking and tree-level masses;
- vertex atlas.

The audit checks unique identifiers, resolved field and sector references, electric-charge conservation, operator dimensions, declared conjugate pairs, recognized couplings, finite parameters, the neutral mass matrix, a massless tree-level photon, mixing-matrix orthogonality, and explicit gauge/source boundaries.

This is a finite structural and tree-level visualization. It does not calculate scattering amplitudes, loop corrections, a path integral, a nonperturbative continuum construction, or a Millennium Prize theorem.
